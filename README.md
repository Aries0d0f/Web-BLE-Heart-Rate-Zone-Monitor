# Web BLE Heart Rate Zone Monitor
Connect your BLE Heart Rate device and support advanced fitness features, like, Heart Rate Zone.

- Demo site: https://hrz.aries0d0f.me

## Develop
- Clone repo
- Install dependencies
  ```shell
  pnpm install
  ```
- Start dev server
  ```
  pnpm run dev
  ```
